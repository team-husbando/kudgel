# Kudgel
Kudgel is a Django web app for scheduling teams.
![Kudgel](https://i.imgur.com/86fxogN.png)

## Requirements
- pipenv

## Installation
```bash
$ git clone https://github.com/team-husbando/kudgel
$ cd kudgel
$ pipenv install
$ python manage.py runserver
```
